# Level Up Coaching Website

Personal life coaching website for levelupcoach.me.

## Stack
- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Hosted on GitHub Pages
- Custom domain: levelupcoach.me

## Local Preview
Open `index.html` in any browser.

## Deployment
Push to `main` branch. GitHub Pages auto-deploys from root.

## Contact Form
Wired to `angela@levelupcoach.me` via `mailto:` — opens user's email client with pre-filled subject and body.
